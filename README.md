# Cyberpunk Ramen Arcade

A cyberpunk-themed WordPress block theme generated by [Miles AI](https://bymiles.ai) during a beta test. Built for a developer portfolio site with a neon-drenched, retro-arcade aesthetic.

## Theme Overview

| Detail          | Value                            |
| --------------- | -------------------------------- |
| Theme Name      | Cyberpunk Ramen Arcade           |
| Author          | Miles AI                         |
| WP Version      | 6.0+ (tested up to 6.7)         |
| PHP Version     | 7.4+                            |
| Schema          | WordPress 6.9 theme.json v3     |
| Text Domain     | cyberpunk-ramen-arcade           |

## Architecture

This is a full-site-editing (FSE) block theme with no traditional PHP template hierarchy. All layout is handled through WordPress block markup in HTML template files.

```
.
├── assets/
│   ├── css/
│   │   ├── styles.css          # Main stylesheet (~3,750 lines)
│   │   └── editor-page.css     # Editor-specific overrides
│   ├── fonts/
│   │   ├── chakra-petch/       # Display font (400, 600, 700)
│   │   └── ibm-plex-mono/      # Monospace font (400, 400i, 500, 600)
│   └── js/
│       └── script.js           # Scroll animations, glitch effects, interactions
├── parts/
│   ├── header.html             # Site header with nav
│   └── footer.html             # Site footer
├── templates/
│   ├── front-page.html         # Homepage with hero section
│   ├── single.html             # Blog post template
│   ├── page.html               # Static page template
│   ├── archive.html            # Blog archive with featured card + grid
│   ├── home.html               # Blog index
│   ├── search.html             # Search results
│   ├── 404.html                # Not found page
│   └── index.html              # Fallback template
├── functions.php               # Enqueues, editor setup, JS class injection
├── theme.json                  # Design tokens, presets, block styles
├── style.css                   # Theme metadata header
└── screenshot.png              # Theme screenshot
```

## Design System

### Color Palette

The theme uses a dark base with three neon accent colors:

| Token           | Value                          | Usage                          |
| --------------- | ------------------------------ | ------------------------------ |
| bg-deep         | `#08081a`                      | Primary background             |
| bg-mid          | `#0e0e28`                      | Section backgrounds            |
| bg-card         | `#12122e`                      | Card surfaces                  |
| bg-card-hover   | `#181844`                      | Card hover state               |
| neon-pink       | `#ff2d7b`                      | Primary accent, CTAs           |
| neon-cyan       | `#00e5ff`                      | Secondary accent, links        |
| neon-green      | `#39ff14`                      | Tertiary accent, status        |
| text-warm       | `#f0e6d6`                      | Primary text                   |
| text-dim        | `#9e99b5`                      | Secondary text                 |
| text-mid        | `#bdb6cf`                      | Mid-weight text                |
| border-subtle   | `rgba(122, 117, 146, 0.18)`   | Card/section borders           |
| border-active   | `rgba(0, 229, 255, 0.4)`      | Active/focus borders           |

Each neon color also has a `*-dim` variant for subtle backgrounds.

### Typography

- **Chakra Petch** -- Display headings, navigation, hero text
- **IBM Plex Mono** -- Body text, code, metadata, labels

Font sizes range from `small` (0.8rem) to `gigantic` (fluid: 3.2rem - 9rem). Fluid typography is enabled.

### Spacing Scale

Uses WordPress preset spacing tokens: `20` (0.5rem) through `65` (8rem).

### Shadows & Gradients

13 shadow presets for neon glow effects (pink, cyan, green variants with inner/layered options). 15 gradient presets covering linear overlays and radial neon glows.

## Interactive Features (script.js)

- **Header scroll effect** -- Adds `.is-scrolled` class after 100px scroll
- **Smooth scrolling** -- Hash link navigation with nav offset
- **Scroll animations** -- IntersectionObserver-driven `.animate-on-scroll` reveal
- **Glitch effect** -- Random text displacement on hero headline
- **Blog card hover** -- Scanline sweep animation
- **Typing cursor** -- Simulated cursor blink on hero tag
- **Stats counter** -- Animated number counting on about section
- **Active nav** -- Highlights current section in navigation
- **Tech stack glow** -- Random neon color on hover

## Development

### File Responsibilities

| File               | Purpose                                                 |
| ------------------ | ------------------------------------------------------- |
| `theme.json`       | Design tokens, presets, block defaults, custom CSS       |
| `styles.css`       | All component styles, animations, responsive, WP blocks  |
| `editor-page.css`  | Block editor overrides, full-width editing, hidden elements |
| `functions.php`    | Asset enqueuing, editor setup, page title removal         |
| `script.js`        | All interactive behavior and scroll-driven animations     |

### Editor Integration

- `miles-editor-hidden` class hides decorative elements (overlays, glow divs) in the block editor
- `editor-page.css` strips padding/margins for full-width editing
- Custom CSS in theme.json handles post editor title styling
